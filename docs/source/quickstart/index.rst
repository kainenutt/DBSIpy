Quickstart
==========

Show help
---------

.. code-block:: bash

   DBSIpy --help

Run from a config
-----------------

.. code-block:: bash

   DBSIpy run --cfg_path PATH/TO/CONFIGURATION.ini

Notes
-----

- If you omit ``--cfg_path``, DBSIpy may launch a GUI flow (Tkinter).
- Shipped config templates live under ``dbsipy/configs/``.
