Introduction
============

What is DBSIpy?
---------------

DBSIpy is a research-oriented diffusion MRI analysis package.
It supports DBSI, DBSI-IA, DTI, and NODDI pipelines driven by configuration files.

Where to start
--------------

- If you want to run an analysis quickly, see :doc:`/quickstart/index`.
- If you need details on config fields and CLI options, see :doc:`/reference/index`.
